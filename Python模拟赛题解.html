题目:我跌跌撞撞奔向你~

by wmh

题目背景

小t和小q迎来了美好的爱情,现在身处山脉两侧的两人需要汇合,出于显而易见的原因,这次应该是小t去找小q.

题目描述

这个山脉被分成n段,每段有一个相对海拔高度hih_ihi​,小t所处的位置h0=0h_0=0h0​=0,他的小q在第n段,他向她前进.

小t从第i段走到第i+1段时,会有∣hi+1−hi∣|h_{i+1}-h_i|∣hi+1​−hi​∣的 跌跌撞撞 值.

但是邪恶的小l获得了一个机会:他可以在路上安置m个炸弹(m<n)(m<n)(m<n)炸弹可以把小t炸回上一个小段,并且造成 422点 跌跌撞撞 值,每段至多安置一个炸弹且小l不能在第0段安置炸弹.

输入格式

第一行两个整数n,m,含义见题面
第二行一行n个整数hih_ihi​

输出格式

一行一个整数,表示答案

输入样例

5 3
1 2 3 4 6

输出样例

1276

数据范围

1≤m<n≤2×1051 \leq m < n \leq 2 \times 10^51≤m<n≤2×105

题解

这n个数使得道路有n小段,根据题意可以计算出每一小段的跌跌撞撞值.

随后这几个炸弹的用处除了直接加422点跌跌撞撞值,还有的就是让小t重走一段路.

那么对每段路的跌跌撞撞值从大到小排序,选择前m个即可

提示

对于 60%60\%60%的数据，保证 1≤n≤2501 \leq n \leq 2501≤n≤250

对于 100%100\%100%的数据 保证 1≤n≤30001 \le n \le 30001≤n≤3000

题解

挂个连接: https://www.luogu.com.cn/blog/WangManhe/Solution-LosingAllHopeOfPrimaryFunction

一点基础知识

考虑坐标系内的两条直线:
l1:y1=k1x+b1l_1:y_1=k_1x+b_1l1​:y1​=k1​x+b1​
l2:y2=k2x+b2l_2:y_2=k_2x+b_2l2​:y2​=k2​x+b2​
有两种可能：

  1. 两直线平行，此时k1=k2k_1=k_2k1​=k2​，两直线没有交点
  2. 两直线相交，
    若b1≠b2b_1 \neq b_2b1​=b2​此时两直线交于(k1−k2b1−b2,k1×k1−k2b1−b2+b1)(\frac{k_1-k_2}{b_1-b_2},k_1 \times \frac{k_1-k_2}{b_1-b_2}+b_1)(b1​−b2​k1​−k2​​,k1​×b1​−b2​k1​−k2​​+b1​)
    若b1=b2b_1 = b_2b1​=b2​

大体思路

考虑绘制和擦除时记录对答案的更改，显然，一条直线会把它穿过的每一个区域分成两个区域，我们注意到，一条直线穿过的区域数等于交点数+1,删除时同理,直接把当前的答案加减穿过的区域数即可.

我们通过set来统计本质不同的交点，然后记录答案即可。

题目: 回文复文回

by yyc

codeforces上的题的翻译

挂链子:https://codeforces.com/problemset/problem/1832/A

题目描述

回文是一个字符串，从左到右和从右到左读相同。例如，"abacaba"，"aaaa"，"abba"，"racecar"是回文。

给你一个字符串s由小写拉丁字母组成。字符串s是一个回文。

你需要判断是否可以重新排列其中的字母以获得另一个回文（不等于给定的字符串s）。

输入格式

每个数据的唯一行包含一个字符串s，s的长度为L（2≤L≤50）

输出格式

对于每个测试用例，如果可以重新排列给定字符串中的字母以获得另一个回文，请输出"YES"，否则请输出"NO"。

样例输入1:

codedoc

样例输出1：

YES

样例输入2：

aabaa

样例输出2：

NO

题解

设len为字符串的长度

如果前⌊len2⌋\lfloor \frac{len}{2} \rfloor⌊2len​⌋个字符都相同，那就不可能得到新的回文字符串。否则，就会有至少两个不同字符，可以把交换它们（以及字符串右半部分与它们对称的字符），并获得与给定字符不同的回文。
